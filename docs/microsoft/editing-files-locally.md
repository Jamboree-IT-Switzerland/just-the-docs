# Editing Files Locally
 
Your Microsoft 365 licence does **not** include the local Office desktop apps (Word, Excel, PowerPoint). Files stored in SharePoint, Teams or OneDrive can be edited directly in the browser via Office Web Apps – no local installation needed.
 
If you prefer a local app, we recommend **ONLYOFFICE Desktop Editors** – a free, open-source alternative that is fully compatible with `.docx`, `.xlsx`, and `.pptx` files:
 
[Download ONLYOFFICE Desktop Editors](https://www.onlyoffice.com/de/download-desktop){: .btn target="_blank" rel="noopener noreferrer"}
